# Frontend - Mini Service de Messages

Ce dossier contient l'interface utilisateur de l'application de messagerie.

## Contenu
- `index.html` : page principale
- `style.css` : styles (mode clair/sombre)
- `script.js` : logique d'affichage et d'interaction avec le back

## Lancement
Ouvrir `index.html` dans un navigateur web.

## Fonctionnalités
- Affichage de la liste des messages
- Envoi de nouveaux messages
- Rafraîchissement manuel
- Changement de thème (clair/sombre)

Le front communique avec le back via des requêtes HTTP (fetch).
